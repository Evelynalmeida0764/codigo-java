# Sistema de Monitoramento de Vasos

Este é um programa simples que permite ao usuário verificar o estado de um vaso de plantas e determinar se ele precisa de rega com base nos dados recebidos de um dispositivo Arduino. O programa oferece duas opções principais:

1. Listagem de Dados dos Vasos:
 O usuário pode escolher esta opção para visualizar informações sobre os dados de vasos cadastrados no sistema.

2. Situação Atual de um Vaso:
Esta opção permite ao usuário selecionar um vaso específico e analisar a porcentagem de umidade e sua situação. O sistema verificará se a umidade atingiu um nível crítico e informará se é necessário regar o vaso.

# Como Usar

1. Execute o programa principal dando run na Main.java.

2. Escolha a opção desejada: Listagem de Dados dos Vasos ou Situação Atual de um Vaso.

3. Para a opção "Situação Atual de um Vaso", o sistema fornecerá informações detalhadas sobre o estado do vaso escolhido com base nos dados do Arduino.

4. O sistema ajudará o usuário a tomar decisões informadas sobre a rega com base na situação da umidade.

5.Para "Listagem de Dados dos Vasos" o sistema mostrará os dados armazenados de determinado período sobre as situações dos vasos.
